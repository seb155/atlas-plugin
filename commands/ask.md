# /ask — Query the AXOIQ Knowledge Engine

Ask any question about the project history, decisions, architecture, or timeline.
Searches across 7 repos, 3,800+ events, memory files, plans, and lessons.

**Usage**: `/atlas ask <question>`

Invoke Skill 'knowledge-engine'.

ARGUMENTS: $ARGUMENTS

Examples:
- `/atlas ask "Why did we choose Authentik over Authelia?"`
- `/atlas ask "What happened during the SSO migration?"`
- `/atlas ask "Show me the architecture evolution"`
- `/atlas ask "Which repos were active on March 25?"`
- `/atlas ask "What are the top themes this month?"`
- `/atlas ask stats` — Show knowledge engine statistics
- `/atlas ask timeline --since 2026-03-20` — Recent timeline
